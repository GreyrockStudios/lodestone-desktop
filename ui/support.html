General
What is Lodestone? ▼
Lodestone is a standalone AI agent engine that gives any LLM persistent memory, self-improvement loops, proactive scheduling, and 39 built-in tools. It's open source (MIT), written in TypeScript, and works with any OpenAI-compatible model — cloud or local. There's also Lodestone Desktop — a native macOS/Windows app with a built-in chat UI, interactive Canvas for artifacts (React, Python, Mermaid, charts, tables), and 38 desktop+server tools wired end-to-end.
Is Lodestone free? ▼
Yes. The Community edition is MIT-licensed and completely free — forever. It includes all 39 tools, the full memory system, self-improvement engine, and community support.
✓ Community (free, MIT): 39 tools, full memory, self-improvement, 5 identity templates, Telegram + Webchat, community support.
✓ Pro ($197 one-time): Everything in Community + Discord & Email channels, MCP integration, priority email support, setup wizard, automation recipes, commercial license.
✓ Enterprise ($997 one-time): Everything in Pro + 5-seat team license, custom tool development, dedicated support channel, on-premise deployment, SLA with 1-hr response.
Which LLMs does Lodestone support? ▼
Any model that speaks the OpenAI API format. This includes OpenAI (GPT-4, GPT-3.5), Anthropic via proxy, Ollama (local models), LM Studio, Together AI, Groq, and more. You configure the provider in lodestone.config.yaml.
Getting Started
How do I get started? ▼ 
npx lodestone init --template developer
cd my-agent
lodestone start

That's it. Pick a template (developer, business, creative, researcher, or general), and your agent boots up with identity, memory, and 39 tools ready to go.

Can I run Lodestone in Docker? ▼
Yes. Lodestone is Docker-ready. Clone the repo, set your environment variables, and docker compose up. The Dockerfile and docker-compose.yml are included in the repo.
How do I customize my agent's personality? ▼
Edit the markdown files in your agent's workspace: SOUL.md (personality and tone), IDENTITY.md (who it is), USER.md (about you), RULES.md (behavioral constraints), and HEARTBEAT.md (what it does proactively). No code needed — just markdown.
Features
How does memory work? ▼
Lodestone has three memory layers: Vector recall (semantic search over everything the agent has learned), Curated wiki (structured knowledge the agent maintains and cross-links), and Session scratch (short-term context for the current conversation). Ask something Tuesday, get the answer instantly Thursday. Knowledge compounds over time.
What is self-improvement? ▼
The self-improvement engine runs prediction journals, pre-mortems, drift detection, and calibration loops. The agent tracks its own accuracy over time, learns from mistakes, and synthesizes new skills. It literally gets better at its job the more it works.
What channels does Lodestone support? ▼
Community edition: Telegram and Webchat. Pro edition: adds Discord and Email (M365/IMAP). Desktop app: built-in chat UI with Canvas for interactive artifacts. All channels share the same agent — your Lodestone talks to you wherever you work.
What is Lodestone Desktop? ▼
Lodestone Desktop is a native macOS and Windows app that gives you a beautiful chat UI, Canvas for interactive artifacts (live React components, Python output, Mermaid diagrams, charts, tables, SVG), and 38 desktop + server tools wired end-to-end. It connects to your own Lodestone server — no data flows through us. Available for macOS (Apple Silicon & Intel) and Windows.
What is Canvas? ▼
Canvas is Lodestone's interactive artifact system, built into the Desktop app. When the AI generates something visual or interactive, it renders in a live side panel — not a static code block. Canvas supports: React (live components like calculators, forms, dashboards), HTML (full HTML/CSS/JS), Python (runs via Pyodide in-browser), Mermaid (flowcharts, sequence diagrams, class diagrams), Charts (bar, line, pie from JSON spec), Tables (sortable, filterable data), and SVG (custom graphics). Because Canvas runs inside the desktop app, it has access to real filesystem and tool capabilities — not just a sandboxed preview.
Does my agent work when I'm not talking to it? ▼
Yes. The heartbeat system lets you define proactive behaviors — health checks, morning briefings, scheduled tasks, and dream mode (offline consolidation). Your agent thinks even when you're not watching.

Troubleshooting
I found a bug. How do I report it? ▼
The best way is to open a GitHub issue. Include your OS, Node version, and steps to reproduce. If you prefer, you can also use the contact form below, email support@greyrockstudios.com, or join our Discord community.
My agent isn't responding. What do I do? ▼
  1. Check that your model API key is set in the config or environment.
  2. Verify the model endpoint is reachable: curl https://api.openai.com/v1/models
  3. Check Lodestone logs: lodestone logs
  4. Restart: lodestone restart
  5. If still stuck, reach out via the contact form below.
Is my data private? ▼
Yes. Lodestone runs locally on your machine (or your own server). Your data — memory, wiki, conversations, API keys — never leaves your infrastructure. There's no telemetry, no analytics, no phone-home. The only external calls are to your configured LLM provider.
Desktop app privacy: The Desktop app connects to your own Lodestone server. No data flows through us — not conversations, not memory, not API keys. Your server, your data.
